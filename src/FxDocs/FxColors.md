# Evality Core Color Palette V1 — 18 base colors

This document outlines the frozen light/core color architecture for the Evality application platform.
**Correction Note:** Blue is explicitly locked as the main interactive Hero Brand Action system. Indigo functions as a secondary utility accent tier.
